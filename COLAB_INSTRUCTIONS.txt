# Google Colab Instructions - STL-10 Classification

## 📦 Quick Setup (2 minutes!)

### Step 1: Upload to Google Colab
1. Go to [Google Colab](https://colab.research.google.com/)
2. Click "Upload" and upload `STL10_Classification_Colab.ipynb`
3. **IMPORTANT**: Set Runtime to GPU
   - Runtime → Change runtime type → Hardware accelerator: **GPU**
   - Click Save

### Step 2: Run Everything!
1. Click "Runtime" → "Run all" (or Ctrl+F9)
2. Wait for completion (~15-20 minutes on T4 GPU)
3. Download results at the end

## ✅ What You'll Get Automatically

- **Question 10 Answer**: Test Accuracy
- **Question 11 Answers**: Class-wise accuracy for all 10 classes
- Wandb dashboard with all visualizations
- Confusion matrix and accuracy plots
- Sample predictions (10 correct + 10 incorrect)
- Downloadable model and results
